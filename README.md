# Property-Manager

Use `docker run -e POSTGRES_DB=postgres -e POSTGRES_PASSWORD=postgres -e POSTGRES_USER=postgres -p 127.0.0.1:5432:5432 postgres` to create PostgreSQL container.

To start using the API:
`sbt run`

API documentation is generated by Swagger.

To use Swagger-UI, head to:
http://localhost:9000/api-docs/swagger-ui/index.html?url=http://localhost:9000/api-docs

Run the evolutions!
